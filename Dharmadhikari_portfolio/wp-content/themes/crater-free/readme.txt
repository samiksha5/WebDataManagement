== Description ==
A WordPress theme suitable for all types of small businesses. 3 prebuilt demos ready to import. More Information here: https://www.spiraclethemes.com/crater-free-wordpress-theme/



== Frequently Asked Questions ==

= Installation =
	
1. In your admin panel, go to Appearance -> Themes and click the 'Add New' button.
2. Type in Crater Free in the search form and press the 'Enter' key on your keyboard.
3. Click on 'Install' button to install and 'Activate' button to use.

= Where can I find the documentation of this theme?

1. Go to https://www.spiraclethemes.com/crater-documentation/
